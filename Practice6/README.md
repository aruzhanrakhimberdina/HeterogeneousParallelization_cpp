## Контрольные вопросы

### 1. Какие основные типы памяти используются в OpenCL?
В OpenCL используется несколько типов памяти. Глобальная память применяется для хранения входных данных и результатов вычислений и доступна всем потокам, но работает относительно медленно. Локальная память используется потоками внутри одной рабочей группы и обеспечивает более быстрый доступ к данным. Приватная память принадлежит одному потоку и используется для временных переменных. Константная память предназначена для хранения данных, которые не изменяются во время выполнения программы.

### 2. Как настроить глобальную и локальную рабочую группу?
Глобальная рабочая группа задает общее количество потоков, которые выполняются параллельно. Локальная рабочая группа определяет, сколько потоков объединяются в одну группу и работают совместно. Эти параметры указываются при запуске ядра. Обычно глобальный размер выбирается так, чтобы обработать все элементы задачи, а локальный размер подбирается с учетом архитектуры устройства.

### 3. Чем отличается OpenCL от CUDA?
CUDA является технологией компании NVIDIA и предназначена только для ее видеокарт. OpenCL представляет собой открытый стандарт и может использоваться на различных устройствах, включая CPU и GPU от разных производителей. CUDA чаще обеспечивает более высокую производительность на GPU NVIDIA, а OpenCL отличается универсальностью и переносимостью.

### 4. Какие преимущества дает использование OpenCL?
OpenCL позволяет использовать один и тот же код на разных аппаратных платформах. Он поддерживает гетерогенные вычисления, дает гибкость в выборе устройства и позволяет масштабировать вычисления при увеличении объема данных. Благодаря этому OpenCL подходит для решения широкого круга вычислительных задач.
