## Контрольные вопросы

### 1. В чём разница между редукцией и сканированием?

Редукция и сканирование выполняют разные задачи. Редукция сводит весь массив к одному итоговому значению, например к сумме всех элементов или максимальному значению. В результате работы редукции получается одно число.

Сканирование, напротив, вычисляет накопленный результат для каждого элемента массива. В этом случае результатом является новый массив той же длины, где каждый элемент зависит от предыдущих. Таким образом, основное отличие заключается в том, что редукция возвращает одно значение, а сканирование возвращает массив значений.

### 2. Какие типы памяти CUDA используются для оптимизации редукции и сканирования?

Для оптимизации редукции и сканирования в CUDA чаще всего используется разделяемая память. Она обеспечивает быстрый доступ для потоков внутри одного блока и позволяет эффективно выполнять промежуточные вычисления.

Глобальная память используется для хранения исходных данных и окончательных результатов. Локальная память применяется для временных переменных каждого потока и обычно не играет ключевой роли в оптимизации, но используется для хранения промежуточных значений.

### 3. Как можно оптимизировать префиксную сумму на GPU?

Префиксную сумму на GPU можно оптимизировать, разделив вычисления на несколько этапов. Сначала префиксная сумма вычисляется внутри каждого блока потоков с использованием разделяемой памяти. Затем вычисляются суммы блоков, которые используются для формирования смещений. Эти смещения добавляются ко всем элементам соответствующих блоков.

Также важно минимизировать количество обращений к глобальной памяти и использовать разделяемую память для основных вычислений. Такой подход позволяет существенно повысить производительность по сравнению с наивной реализацией.

### 4. Приведите пример задачи, где применяется сканирование.

Сканирование широко используется в задачах параллельной сортировки, например в сортировке по разрядам, где требуется определить позиции элементов. Также префиксная сумма применяется при построении гистограмм, обработке изображений и в алгоритмах распределения данных, где необходимо вычислять накопленные значения или индексы.

Во всех этих задачах сканирование играет ключевую роль, так как позволяет эффективно выполнять операции, зависящие от предыдущих элементов массива.
